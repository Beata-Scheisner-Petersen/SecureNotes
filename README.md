## General information
This project is a console‑based notes application built in Java using a clean MVC architecture. It allows users to register, log in, manage personal notes, and update their password. The application uses Maven for dependency management and project structure, ensuring a modular and maintainable codebase. Business logic is handled in dedicated service classes, while repositories manage data persistence through MySQL. All components are wired together in App.java using manual dependency injection to keep the system flexible, testable, and easy to extend.
